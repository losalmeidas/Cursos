Desafio 001

Trabalhando com Listas.

Listas Ordenadas

As listas ordenadas seguem uma ordem, "ol".

Ou podem ser sem ordem definida, "ul"

Exemplos

Lista Ordenada

Alfabeto Grego:

  1. alfa
  2. beta
  3. gama
  4. delta

Receita de bolo:

  • farinha
  • ovos
  • açucar
  • leite
